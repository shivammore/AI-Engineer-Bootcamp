# AI Engineer Bootcamp Roadmap

## Project Astra

**Student:** Shivam More

**Mentor:** ChatGPT (Senior AI Engineer)

**Mission:** Become a Production-Ready Agentic AI Engineer capable of designing, building, deploying, and maintaining enterprise-grade AI systems.

**Duration:** 180 Days

**Capstone Project:** AI Workspace

---

# Learning Philosophy

This bootcamp follows one core principle:

> **Understand the problem before choosing the technology.**

We will always learn in this order:

Business Problem
→ Requirements
→ Architecture
→ Components
→ Technology
→ Implementation
→ Deployment
→ Monitoring
