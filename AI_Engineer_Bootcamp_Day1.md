# Day 1

## Concepts Learned

### What is an AI Engineer?

An AI Engineer builds complete AI systems around large language models,
including APIs, databases, memory, vector stores, deployment,
monitoring, and user interfaces.

### What is Agentic AI?

Instead of producing a single response, an AI Agent:

-   Thinks
-   Plans
-   Chooses tools
-   Executes actions
-   Observes results
-   Retries if necessary
-   Produces an answer

### AI Evolution

Transformers → LLMs → Chatbots → RAG → AI Agents → Multi-Agent Systems
